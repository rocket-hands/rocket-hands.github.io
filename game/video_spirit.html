Video Spirit

Video Spirit is a rudimentary game for March's One Game a Month.

The theme is 'neon' and I decided to use an old running animation I made years ago. And add Neon.

Using Unity again and playing around with mobile deployment and onscreen controls this time.

The game currently has no sound and only a rudimentary scoring system. It's still kind of fun for a little bit but needs work to be considered finished. I'm not sure if I'll do more work on this game, at least probably not right away.
